# aoe2 campaign parser
`CpxFile` class handles campaign files, which extensions are `*.cpn`, `*.cpx`, `*.cpx2`, `*.aoe2campaign`

`ScxFile` class handles Scenario files.

[Online Demo](https://lab.bangbang93.com/aoe2-campaign)
